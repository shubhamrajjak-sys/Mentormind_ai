

# Study Buddy AI – Landing Page Plan

## Overview
A modern, polished SaaS landing page for an AI-powered education platform. Fully responsive, with smooth animations and a soft blue/white design system.

---

## Design System
- **Colors**: Soft blue primary (#4F7DF3), white, light grey background (#f8f9fc)
- **Typography**: Inter font, clean hierarchy
- **Style**: Rounded corners, soft shadows, spacious layout
- **Animations**: Fade-in on scroll, hover scale effects, smooth transitions

---

## Page Sections

### 1. Header / Navigation Bar
- Logo text "Study Buddy AI" on the left
- Navigation links: "Study Buddy AI", "Blog"
- Right side: "Sign Up" (blue button) and "Sign In" (light grey button)
- Sticky header with subtle shadow on scroll

### 2. Hero Section
- Large bold heading: "Study Buddy AI"
- Subheading about AI learning assistant
- Two CTA buttons: "Get Started" (primary) and "Sign In" (secondary)
- Right side: SVG/CSS illustration of student with AI elements (globe, books, science icons) in blue flat vector style

### 3. AI-Powered Learning Section
- Section label: "Experience the future of education"
- Heading: "AI-Powered Learning"
- Description paragraph
- 4 feature cards with circular blue icons:
  - Personal AI Tutor
  - Smart Flashcard Creator
  - Comprehensive Notes Assistant
  - Adaptive Quiz Builder

### 4. Learn Faster Section
- Centered subheading: "StudyBuddy can help you"
- Large title: "Learn Faster and Better"
- 3 feature columns with icons, titles, descriptions, and "Learn more →" links:
  - Unlimited Learning Potential
  - Smart Teaching Methods
  - Adapts to Your Expertise

### 5. Learn From Any Format Section
- Blue label badge: "Upload all kinds of learning materials"
- Large title: "Learn From Any Format"
- Left side: 4 text features with icons (Notes, Research Papers, Videos, Web Knowledge)
- Right side: Illustrative graphic of laptop with documents and charts

### 6. Footer
- Dark background
- Copyright: © 2026 Study Buddy AI
- Clean, minimal design

---

## Responsive Behavior
- All sections stack vertically on mobile
- Navigation collapses to hamburger menu
- Hero illustration moves below text on small screens
- Feature columns become single column on mobile

## Interactions
- Fade-in animations as sections scroll into view
- Hover effects on buttons (scale + shadow)
- Hover effects on feature cards (lift + shadow)
- Smooth scroll between sections
